# GAIA-Transfer Human-Audited Package

This package contains project-owner-audited obligation-transfer labels for 100 GAIA validation rows and 100 GAIA test rows. It evaluates obligation/profile transfer and harness selection only; it does not report GAIA answer-level correctness.

Use `gaia_transfer_200_human_audited.jsonl` for the combined 200-row source, or the split files for validation/test analysis. Row-level metadata includes `source_dataset`, `source_config`, `source_split`, and `source_task_id`.
